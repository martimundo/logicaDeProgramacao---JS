# logicaDeProgramacao---JS
Curso de Lógica de Programação e Algoritmo com JavaScript

#Codigos:

Os códigos aqui apresentados foram criados com base no livro Lógica de Programa e Algorimos com JavaScript do Professor Edécio Fernando Lepsen.
Cada Pasta guarda um capitulo do Livro. Foi Estruturado capitulo por capitulo, exemplo por exemplo.

Ao todo são 13 capitulos que iniciam desde a criação de variaveis a criação de aplicação com Back-end utilizando o Express.

##O Livro possui uma leitura bem simples e objetiva.
